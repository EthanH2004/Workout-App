Visual
Design System

The full kit of parts, made visible. Every token and component on its own, the states split out, and a few assembled sample screens. These are examples of the pieces to use — not the final design of any page.

Dark only · stark & graphic · true-neutral grays · one warm-orange accent · Archivo · big tabular numbers · snappy motion. Everything here is dark-on-dark by design; each component sits on a darker app-canvas panel so you can see how it separates by lightness.
